# socialLLM
Open codebase for socialLLM. (Currently anonymized, will be fully prepared after the review process.)
- File description
  - socialLLM.py: Utilities and pre-defined methods.
  - socialLLM.ipynb: Notebook for conducting Stage I and II of the social trials.
  - socialLLM_figure.ipynb: Notebook for drawing figures.
- The dataset we used (ConceptARC) is fully open and available at github.com/victorvikram/ConceptARC.
- User-friendly manuals and instructions will be ready soon.
